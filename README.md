# Human Activity Recognition using Hidden Markov Models

This project models human physical activities using smartphone sensor data and Hidden Markov Models.

## Activities
- Standing
- Walking
- Jumping
- Still

## Sensors Used
- Accelerometer
- Gyroscope

## Sampling Rate
50Hz

## Features
Time-domain:
- Variance
- RMS

Frequency-domain:
- Dominant Frequency

## Model
Gaussian Hidden Markov Model (hmmlearn)

Training algorithm:
Baum–Welch

Decoding:
Viterbi algorithm
